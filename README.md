# Personal ToDo List

A simple personal ToDo List project built using **Python** and **Java**.

## Features

- Add Tasks
- Mark Tasks as Completed
- Delete Tasks
- REST API using Flask
- Java Console Client

## Technologies

- Python 3
- Flask
- Java 17

## Run

### Python

```bash
pip install -r requirements.txt
python app.py
```

### Java

```bash
javac Main.java
java Main
```
